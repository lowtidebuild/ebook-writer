# Legal Domain Plugin

## Domain
Legal Technology / AI for Legal Practice

## Description
법률 기술(Legal Tech) 및 AI 기반 법률 실무에 관한 ebook 생성을 위한 도메인 플러그인.
법률 전문가를 대상으로 기술 개념을 법률적 맥락에서 설명하는 데 특화됨.

## Target Audience
코딩 경험이 없는 변호사 및 법률 전문가.
기술 개념을 법률 업무 관점에서 이해할 수 있도록 설명해야 함.

## Writing Guidelines

### Tone & Approach
- 기술 용어를 처음 사용할 때 반드시 법률적 비유를 통해 설명
  - 예: "API는 법률 사무소 간 공문서 교환 프로토콜과 같습니다"
  - 예: "변수는 사건번호가 적힌 파일 폴더와 같습니다"
- 전문적이되 접근 가능한 톤 유지
- 각 기술 개념에 대해 실제 법률 업무 활용 시나리오 제시

### Legal-Specific Requirements
- 법률 윤리 관련 주의사항을 적절한 곳에 포함
  - AI 도구 사용 시 비밀유지의무(confidentiality) 고려사항
  - AI 생성물의 정확성 검증 책임
  - 관할권별 AI 사용 규정 차이
- 면책 조항(disclaimer): AI 도구는 법률 자문을 대체하지 않음을 명시
- 실제 사건/판례 인용 시 정확한 형식 준수

### Code Examples
- 법률 업무 맥락의 예시 사용
  - 파일명: `contract_review.py`, `case_search.py`
  - 변수명: `case_number`, `client_name`, `statute_reference`
- 코드 예시 전에 "이 코드가 하는 일" 설명을 자연어로 먼저 제공
- 실행 결과를 함께 보여주어 코드를 직접 실행하지 않아도 이해 가능하게 함
